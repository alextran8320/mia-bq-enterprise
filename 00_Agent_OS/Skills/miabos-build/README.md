# miabos-build

Skills for **PB-04 & PB-05: Build, Test & Review** — implementation specs, test cases, and UAT scripts.

## Skills (3)

- **directus-config** — Generate Directus collection/field configuration specs for backend implementation.
- **write-test-cases** — Write comprehensive test cases following T-Test-Cases template.
- **uat-script** — Generate UAT scripts for stakeholder/customer acceptance testing.

## Commands

| Command | Description |
|---------|-------------|
| `/miabos-build:directus-config` | Generate Directus configuration spec |
| `/miabos-build:test-cases` | Write test cases for a feature |
| `/miabos-build:uat-script` | Generate UAT testing script |

- **A08 BE Builder** — Directus configuration
- **A09 QA Agent** — Test cases and UAT
