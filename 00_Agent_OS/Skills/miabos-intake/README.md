# miabos-intake

Skills for **PB-01: Discovery & Intake** — processing raw inputs into structured, actionable artifacts.

## Skills (3)

- **process-mom** — Process raw MOM transcript into structured decisions, action items, and KB rules. Uses T-MOM template.
- **create-project** — Initialize a new project workspace from `01_Projects/_template/` with proper linking and metadata.
- **intake-feedback** — Process customer feedback, feature requests, or bug reports into categorized backlog items.

## Commands

| Command | Description |
|---------|-------------|
| `/miabos-intake:process-mom` | Process a MOM transcript file or pasted text |
| `/miabos-intake:create-project` | Create a new project workspace |
| `/miabos-intake:intake-feedback` | Process customer feedback into backlog items |
